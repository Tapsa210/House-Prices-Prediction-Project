# House Prices Prediction Project 🏡📊

## Description
This project predicts house sale prices using advanced regression techniques. The goal is to accurately estimate house prices based on multiple features such as area, neighborhood, age, and more. 

Key aspects of this project include:
- Data cleaning and handling missing values
- Feature engineering (Total Square Footage, House Age, interaction features)
- Encoding categorical variables with One-Hot Encoding
- Outlier detection and removal
- Regression models with regularization: **Lasso, Ridge, ElasticNet**
- Hyperparameter tuning using **GridSearchCV**
- Model evaluation using **RMSE** and **R²**
- Model blending for improved predictions

This project is based on the Kaggle competition: [House Prices: Advanced Regression Techniques](https://www.kaggle.com/competitions/house-prices-advanced-regression-techniques).
